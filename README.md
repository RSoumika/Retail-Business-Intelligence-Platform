# Retail Business Intelligence Platform

## Project Overview

This project analyzes retail sales data from the Superstore dataset to identify business insights related to revenue, profit, customers, products, regions, and discount strategies.

The objective is to support data-driven decision making using SQL, Python, and Power BI.

---

## Tools Used

- SQL (MySQL Workbench)
- Python
- Pandas
- Matplotlib
- Power BI
- Git & GitHub

---

## Project Workflow

1. Data Cleaning using Python (Pandas)
2. Data Analysis using SQL
3. Business Insight Generation
4. Dashboard Development using Power BI
5. Reporting and Visualization

---

## Key Business Insights

- Total Revenue: 2.27M
- Total Profit: 282.86K
- Total Orders: 4,931
- Profit Margin: 12.45%

### Regional Performance

- West Region generated the highest revenue and profit.
- South Region generated the lowest revenue.
- Central Region showed lower profitability despite strong sales.

### Category Performance

- Technology was the highest-performing category.
- Furniture generated strong revenue but low profit.
- Office Supplies delivered better profitability than Furniture.

### Customer Insights

- Tamara Chand was among the most profitable customers.
- High sales do not always result in high profit.

### Product Insights

- Canon imageCLASS 2200 Advanced Copier was the most profitable product.
- Several high-revenue products generated losses.

### Discount Analysis

- Discounts above 30% frequently resulted in losses.
- Profitability declined significantly as discount levels increased.
